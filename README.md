# CTEDS Digital Menu 
Projeto final Capacitação Tecnológica em engenharia e desenvolvimento de software, oferecido pela USP

### Pra quem é esse app?
- Para você que quer vender QUALQUER COISA e precisa de um cardápio ou catálogo prático, organizado e de fácil entendimento.

### Funcionalidades
- Facilidade na edição e criação de seus produtos;
- O cliente acessa o seu cardápio pelo celular, com uma apresentação agradável e interativa;
- Sem informações erradas, tudo fica centralizado em sua máquina com segurança e confiabilidade;
- Baixo custo de implementação e sem complicações para usar.

## Para usar o programa
- Clone o repositório
- Abra a pasta no Visual Studio
- Troque para a branch dev
- Abra o projeto CTEDSMenuDigital.sln
- Execute pelo Visual Studio

### Atenção, programa em fase de desenvolvimento.
